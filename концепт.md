# Концепт-документ

**Название проекта:** Система семантического поиска по локальному хранилищу медиафайлов  
**Дата создания:** 30.06.2026  
**Версия:** 1.0  
**Статус:** В разработке  

---

## Краткое описание

Локальная система поиска по медиафайлам на компьютере пользователя.  
Пользователь вводит запрос своими словами, а система находит подходящие изображения, видео и аудиофайлы — даже если в названии файла нет нужных слов.

---

## Проблема

Обычный поиск по папкам работает только по имени файла. Это неудобно, потому что:

- большинство файлов имеют автоматические названия вроде `IMG_4821.jpg`
- пользователь помнит содержание файла, но не помнит его название
- найти нужный файл в большом архиве занимает много времени

---

## Идея

Сделать систему, которая анализирует содержимое медиафайлов и позволяет искать их по смыслу текстового запроса.

Примеры запросов:

- «фото с моря»
- «видео с котом»
- «запись лекции про нейросети»

---

## Цель

Разработать прототип локальной системы семантического поиска по медиафайлам, которая умеет индексировать файлы, анализировать их содержимое и находить подходящие результаты по текстовому запросу.

---

## Задачи

1. Изучить подходы к семантическому поиску
2. Выбрать технологии для обработки медиафайлов
3. Реализовать индексацию локального хранилища
4. Реализовать извлечение признаков из медиафайлов
5. Создать механизм поиска по текстовому запросу
6. Сделать простой пользовательский интерфейс
7. Проверить работу системы на тестовом наборе файлов

---

## Целевая аудитория

- Обычные пользователи с большим количеством фото и видео
- Студенты с большим архивом записей лекций
- Контент-мейкеры и монтажёры
- Исследователи и разработчики, работающие с медиаархивами

---

## Сценарий использования

1. Пользователь выбирает папку на компьютере
2. Система сканирует медиафайлы в этой папке
3. Для каждого файла создаётся векторное представление
4. Пользователь вводит текстовый запрос
5. Система сравнивает запрос с индексом и выводит подходящие результаты
6. Пользователь открывает нужный файл
